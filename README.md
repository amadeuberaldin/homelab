# Homelab

This repository documents my self-hosted homelab environment, where I experiment with infrastructure, networking, VPNs, Docker, and real-world services.

The goal of this project is to learn by building, testing, and documenting practical systems running on my own hardware and VPS infrastructure.

---

## What this homelab includes

- Local Ubuntu server
- VPS acting as a remote hub
- WireGuard VPN infrastructure
- Docker-based services
- Private remote access
- Game server experiments
- Self-hosted development environment

---

## Main Areas

- Linux system administration
- Networking and VPN architecture
- Self-hosted services
- Infrastructure documentation
- Practical experiments and learning

---

## Repository Structure

- `docs/` general documentation
- `network/` network topology and VPN notes
- `services/` services running in the lab

---

## Status

This homelab is actively evolving as I improve the infrastructure, organize services, and document the environment.
