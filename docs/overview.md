# Overview

This homelab is my personal infrastructure environment used to study and practice real-world systems administration, networking, and self-hosting.

## Goals

- Learn infrastructure by building real systems
- Practice Linux server administration
- Build secure remote access through VPN
- Run services in a controlled personal environment
- Document architecture and experiments

## Current Environment

The lab currently includes a local Ubuntu server, a VPS used as a VPN hub, and multiple services running in containers or directly on the host.

This environment is also used for experimentation with self-hosted tools, networking, remote access, and service organization.
